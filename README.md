# ♟️ Chess Game

A fully functional chess game built using **HTML, CSS, and Vanilla JavaScript**. The project implements the core mechanics of chess with an interactive UI, legal move validation, turn-based gameplay, and special chess rules.

## 🚀 Live Demo

https://pari7maheshwari.github.io/Chess-Game/

## ✨ Features

* ♟️ Interactive 8×8 chessboard
* 🎯 Click-to-select and move pieces
* ✅ Legal move validation for all chess pieces
* 🔄 Turn-based gameplay (White & Black)
* ✨ Highlights legal moves for the selected piece
* ⚔️ Piece capturing
* 👑 Check detection
* 🏆 Checkmate detection
* 🤝 Stalemate detection
* 🏰 Castling
* ♟️ Pawn promotion
* ⚡ En passant
* 📱 Responsive design

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/pari7maheshwari/chess-game.git
```

### Navigate to the project

```bash
cd chess-game
```

### Run the project

Open `index.html` in your browser or use the **Live Server** extension in Visual Studio Code.

---

## 🎮 How to Play

1. Click on one of your pieces to select it.
2. Valid moves for the selected piece are highlighted.
3. Click a highlighted square to move the piece.
4. Players alternate turns between White and Black.
5. The game follows standard chess rules, including:

   * Check
   * Checkmate
   * Stalemate
   * Castling
   * Pawn Promotion
   * En Passant
